# Python Mini Projects

This repository contains beginner-friendly Python projects developed to improve programming skills and build a strong foundation for software development and AI.

## Projects Included

### Number Guessing Game
A simple game where the user tries to guess a randomly generated number.

Features:
- Random number generation
- Hints (Too high / Too low)
- Attempt counter

### Simple Calculator
A basic calculator that performs arithmetic operations.

Features:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles division by zero

## Technologies Used

- Python 3
- Built-in modules (`random`)
